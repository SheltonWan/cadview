## 1. 产品简介

**PropOS CAD** 是一款轻量级 CAD 文件查看器，支持在桌面和移动设备上浏览 **DWG** 与 **DXF** 格式的图纸文件。无需安装完整的 AutoCAD，即可快速预览工程图纸，适用于现场查阅、文件审核等场景。

**支持格式**

| 格式 | 说明 |
|------|------|
| `.dwg` | AutoCAD 原生二进制图形格式 |
| `.dxf` | 图形交换格式（Drawing Exchange Format） |

## 3. 启动应用

安装完成后，双击应用图标或从启动器中打开 **PropOS CAD**。

首次启动时，应用会完成内部查看器资源的初始化（通常仅需数秒）。初始化完成后，进入**文件浏览主界面**。

## 5. 打开 CAD 文件

### 方式一：选择文件夹浏览

1. 点击顶部工具栏的 **📂（文件夹）** 图标，或欢迎页中的 **"选择文件夹"** 按钮。
2. 在系统文件选择对话框中，导航到存放图纸的目录并确认。
3. 应用将自动扫描该目录下的所有 `.dwg` 和 `.dxf` 文件，以列表形式呈现。
4. 点击任意文件条目，进入 CAD 查看界面。

> **提示：** 若所选文件夹中没有 DWG/DXF 文件，界面将提示"此文件夹中没有 DWG / DXF 文件"，可点击「选择其他文件夹」重新选择。

### 方式二：直接打开单个文件

1. 点击顶部工具栏的 **📄（文件）** 图标，或欢迎页中的 **"打开文件"** 按钮。
2. 在文件选择对话框中，选择目标 `.dwg` 或 `.dxf` 文件并确认。
3. 应用直接进入该文件的查看界面。

## 6. 查看 CAD 图纸

进入查看界面后，图纸将自动加载并居中显示。

### 基本操作

| 操作 | 桌面端 | 移动端 |
|------|--------|--------|
| 缩放 | 鼠标滚轮 / 触控板双指 | 双指捏合/展开 |
| 平移 | 按住鼠标左键拖动 | 单指拖动 |
| 旋转（3D 图纸） | 按住鼠标右键拖动 | 三指拖动 |
| 重置视图 | 双击空白区域 | 双击空白区域 |

### 重新加载文件

若图纸显示异常，可点击顶部工具栏右侧的 **🔄（刷新）** 图标重新加载当前文件。

## 7. 常见问题

**Q：打开文件后一直显示"正在加载 CAD 文件…"转圈，没有内容显示。**  
A：可能是文件损坏或格式版本过高。请确认文件可以在其他 CAD 工具中正常打开，并尝试另存为较早版本的 DWG 格式（推荐保存为 AutoCAD 2013 或更低版本）。

**Q：选择文件夹后，明明有 DWG 文件，但列表为空。**  
A：请检查文件扩展名大小写是否正确（应为 `.dwg` 或 `.dxf`），并确认文件未被隐藏。应用仅扫描所选目录的**第一层文件**，不递归扫描子文件夹。

**Q：出现"无法启动本地服务"错误提示。**  
A：请尝试重启应用。若问题持续，请检查是否有其他程序占用相关端口，或重新安装应用。

**Q：图纸内容能看到，但文字/字体显示为乱码或方块。**  
A：这通常是 CAD 图纸使用了系统中未安装的自定义字体。图纸查看功能将使用内置替代字体呈现，不影响图形内容的显示。
